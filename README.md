# Spotify Data Analysis

## Overview
This project analyzes Spotify music data to uncover trends and relationships between track features and popularity. The analysis includes data cleaning, feature engineering, exploratory data analysis (EDA), and visualization.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Project Objectives
- Clean and preprocess Spotify dataset
- Handle missing values and duplicates
- Create new features for analysis
- Categorize songs based on popularity
- Visualize relationships between audio features
- Generate insights from the dataset

## Key Insights
- Identified correlations between audio features and popularity.
- Analyzed distribution of song characteristics.
- Segmented tracks into popularity categories.

## Files
- `Spotify.ipynb` - Main Jupyter Notebook containing analysis and visualizations.

## How to Run
1. Clone the repository
2. Install required libraries:
   ```bash
   pip install pandas numpy matplotlib seaborn
   ```
3. Open the notebook:
   ```bash
   jupyter notebook Spotify.ipynb
   ```
